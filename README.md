# Keelson Connector Camera Realsense 

Connector based on https://www.intelrealsense.com/sdk-2/ 

## Quick start

```bash 
# Thermal Camera
python3 bin/main.py --log-level 10 -r rise -e storakrabban -s camera/thermal/0 --ip-address 192.168.3.12 --camera-thermal

```
